# Phantom Lens 👻

Phantom Lens is a prank application designed for April Fool's Day. It disguises itself as a high-tech "ghost scanner" using the device's camera and various fake sensors to "detect entities."

## Features

- **Fake Calibration:** Shows a realistic initialization process to build tension.
- **System Failure Glitch:** The UI starts to glitch and shake before the big reveal.
- **Ultra Jumpscare:**
    - High-impact video jumpscare.
    - Intense red strobe lights.
    - Violent screen jitter and digital noise.
    - Max-intensity haptic feedback (vibration).
    - Layered audio with a terrifying screech.
- **The Reveal:** A "tuff" purple reveal page after the scare to celebrate April Fool's Day.

## How to Install

You can find the pre-built APK for this project here:
`app/build/outputs/apk/debug/app-debug.apk`

To install it on your Android device:
1. Enable "Install from Unknown Sources" in your device settings.
2. Transfer the `app-debug.apk` file to your phone.
3. Open the file and follow the installation prompts.

## Development

Built with:
- **Jetpack Compose** for the UI.
- **CameraX** for the live camera preview.
- **Media3 (ExoPlayer)** for high-performance video jumpscares.
- **Kotlin Coroutines** for precise prank timing.
