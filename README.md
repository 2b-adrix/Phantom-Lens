# 👻 Phantom Lens: The Ultimate Ghost Scanner Prank

**Phantom Lens** is a high-intensity prank application designed for April Fool's Day. Disguised as a sophisticated "paranormal activity detector," it uses psychological tension and technical "glitches" to deliver a heart-pounding jumpscare experience.

---

## 🚀 How it Works (The Prank Flow)

1.  **The Hook:** The app opens to a sleek, minimal camera interface with a "Calibrating Sensors..." overlay.
2.  **Building Tension:** A series of fake technical tasks appear (Magnetometer calibration, Ionizing radiation scan, etc.) to establish credibility.
3.  **The "Glitch":** As the progress bar nears 100%, the UI begins to stutter, turn red, and display "System Overload" errors to make the user lean in closer.
4.  **The Payoff:** A sudden, full-screen jumpscare video triggers with high-impact audio, followed by a "Gotcha!" reveal screen.

---

## ✨ Key Features

-   **Realistic Camera UI:** Uses **CameraX** for a live, high-frame-rate background preview.
-   **Dynamic Glitch Engine:** Custom Compose animations simulate UI corruption and "kernel panics."
-   **Multi-Sensory Scare:**
    -   **Visual:** High-impact video playback using **Media3 ExoPlayer**.
    -   **Atmospheric:** Intense red strobe effects and digital noise overlays.
    -   **Audio:** Layered screeching sound effects (optional/configurable).
-   **The Reveal:** A vibrant, celebratory April Fool's message to break the tension.

---

## 🛠️ Technical Stack

-   **Language:** Kotlin
-   **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
-   **Camera:** [CameraX](https://developer.android.com/training/camerax)
-   **Media:** [Media3 ExoPlayer](https://developer.android.com/guide/topics/media/exoplayer)
-   **Architecture:** MVVM (Model-View-ViewModel) with StateFlow
-   **Concurrency:** Kotlin Coroutines for precise event timing

## 📲 Installation

1.  Navigate to the latest build: `app/build/outputs/apk/debug/app-debug.apk`.
2.  Enable **"Install from Unknown Sources"** on your Android device.
3.  Transfer and install the APK.
4.  Launch "Phantom Lens" and hand it to a friend!
